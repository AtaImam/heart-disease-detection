# ❤️ Heart Disease Prediction System

## 📌 Overview

This project is a **machine learning-based heart disease prediction system** with a simple **Flask web interface**. Users can enter medical and health-related information into a form and receive a prediction about the likelihood of heart disease.

The system uses **two machine learning algorithms**:

* **Logistic Regression**
* **Random Forest**

This project demonstrates how AI and machine learning can be applied in healthcare to support early risk assessment and decision-making.

---

## 🎯 Objectives

* Predict the likelihood of heart disease using patient health data
* Compare the performance of two machine learning algorithms
* Provide a simple web-based interface for user input and prediction
* Demonstrate a practical healthcare application of machine learning

## 🤖 Machine Learning Models

This system uses two different models:

### 1. Logistic Regression

* Implemented as a custom model in the notebook
* Used for binary classification of heart disease risk
* Supports probability-based prediction

### 2. Random Forest

* Implemented using `scikit-learn`
* Used to improve predictive performance through ensemble learning
* Helps compare results with Logistic Regression

---

## 📊 Dataset

The model is trained on heart disease data containing medical attributes such as:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Maximum Heart Rate
* Oldpeak
* Other clinical indicators

## 🌐 Web Application

The project includes a simple Flask-based web interface where users can:

* Enter medical input values
* Select a prediction model

The web interface supports both:

* **Logistic Regression**
* **Random Forest**

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/AtaImam/heart-disease-prediction-system.git
cd heart-disease-prediction-system
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Flask app

```bash
python app.py
```

### 4. Open in browser

Visit:

```text
http://127.0.0.1:5000/
```
